# AuthBlazorServerApp
## Проект по теме: Внедрение авторизации и аутентификации в Blazor Server
Созданы поверх прошлого проекта: механизм аутентификации, страница для входа, функция выхода из системы, механизм авторизации; отключено отображение страниц для незарегистрированных пользователей.

Внедрено сохранение ключа сессии в локальном хранилище браузера (Local Storage), чтобы не было необходимости пользователю вновь входить на сайт после перезапуска браузера.

Добавлена страница регистрации и обеспечена возможность регистрироваться новым пользователям.
